fn midnight(&self) -> Option<i64>

Returns the midpoint between sunset and sunrise. Will return None if ITL can not reliably calculate the necessary values.

Since this function internally uses the sunrise of the current day rather than the day after, the result will be slightly inaccurate.

Note that hours can go over 24, which indicates that midnight occurs the following day.

Note that this value does not represent the end of the Isha time if it ends at "midnight". In Islam, daytime starts at dusk, rather than sunrise.
